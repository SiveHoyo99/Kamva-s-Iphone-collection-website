# Kamva's iPhone Collection

> **Live Demo:** [View Project Here](https://sivehoyo99.github.io/Kamva-s-Iphone-collection-website/)

## Overview
A responsive static website that is developed for marketing reasons for a growing business "Kamva's Iphone Collection ". It displays Iphone products with prices and descriptions. A button (Order) redirecting to WhatsApp is included.

---

## Features
-  Responsive design for all devices
-  Clean and modern UI
-  Fast-loading static pages

---

## Tech Stack
- **HTML5**
- **CSS3**
